# 宸观 BrightEye —— 宸宇护目，智能时长管控护眼伴侣系统

面向长时间用眼人群的**非接触式 AI 视觉健康监测**桌面应用（参赛原型 Demo · v1.10.0）。
普通摄像头实时分析**眨眼频率 / 用眼距离 / 坐姿(颅椎角·高低肩) / 表情情绪**，结合
20-20-20 护眼法则给出分级健康建议；悬浮桌宠「文乃」傲娇陪伴、可接入大模型自然对话，
复盘报告提供雷达图/趋势图/热力图可视化与 AI 行为洞察，极端用眼时触发强制护眼干预。

- 📦 **软件代码与完整文档**：见 [`brighteye/`](brighteye/)（[README](brighteye/README.md) · [CHANGELOG](brighteye/CHANGELOG.md)）
- 🚀 **快速运行**：`python -m brighteye.main --simulate`（离线优先铁律：无摄像头/无大模型也能完整演示）
- 🧠 **大模型（可选增强）**：本地 Ollama `qwen2.5:7b-instruct`（对话/台词）+ `deepseek-r1:7b`（复盘洞察），不可用自动回退离线

> 本仓库仅包含软件代码；商业计划书等竞赛文档与演示打包不在开源范围内。
> 合规说明：本产品仅提供用眼/体态/情绪行为引导与提示，不构成医疗诊断、治疗或用药处方。
