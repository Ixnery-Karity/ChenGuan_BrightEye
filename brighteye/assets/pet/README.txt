桌宠「文乃」galgame 立绘 · 放图说明（路线 B）
============================================

把 AI 生成的透明背景 PNG 放进本文件夹，程序启动时会自动加载并替换掉
代码手绘的兜底形象。文件不存在时，自动回退到内置手绘矢量形象（demo 照常可用）。

画风＝galgame / 视觉小说立绘（平涂赛璐璐上色、动漫脸、正面全身），不是像素风。
出图提示词见同目录《立绘出图提示词_文乃_galgame差分.txt》，复制即用。

【需要的文件名】（必须完全一致，全部小写）+【每张该画什么动作】
  按「担心程度」差分动作——同一个文乃，只换表情和姿势：

  wenna_idle.png    日常 / 待机（必需；缺它则整体回退手绘）
                    → 自然站立、双手自然垂、虎牙微笑。先做这张当"母版"。
  wenna_happy.png   满意 / 开心（用眼习惯好，鼓励）
                    → 双手举起欢呼 / 比心，灿烂笑。
  wenna_blink.png   通用眨眼帧（可选；缺则其余情绪眨眼时保持睁眼，不做突兀切换）
                    → 与 idle 完全同姿势，仅闭眼。
  wenna_pout.png    嘟嘴 / 轻度提醒（出现轻微不良习惯）
                    → 一手叉腰、一手食指上举提醒，鼓腮嘟嘴。
  wenna_angry.png   生气 / 严厉警告（持续不良用眼，严重度高）
                    → 双手叉腰、身体前倾，怒眉训斥。
  wenna_sleepy.png  犯困 / 无人值守（检测不到人脸）
                    → 一手揉眼、微驼背、头边飘 Zzz。

  只放 wenna_idle.png 也能用，其余情绪会统一用 idle 帧。

  【各情绪专属眨眼差分（可选，进阶）】
  想让每种情绪眨眼时仍保持自己的姿势（动作更连贯不突兀），可再放：
    wenna_happy_blink.png  / wenna_pout_blink.png  / wenna_angry_blink.png
  与同名情绪图完全同姿势、仅闭眼即可。缺某张时该情绪眨眼自动回退到
  通用 wenna_blink.png；连通用也没有则该情绪眨眼时保持睁眼。
  （睡眠 sleepy 本身就是闭眼睡颜，不需要、也不会再插入眨眼帧。）

【差分一致性（最重要）】
  6 张必须是「同一个人」：脸型/发型/配色/画风完全一致，只换表情+动作。
  做法：先把 idle 刷到满意当母版，其余用「图生图/参考图」基于母版出，
  锁同一 seed 或同一参考图，一致性最好。详见提示词 txt。

【出图规格建议】
  - 透明背景（alpha 通道）；正面、全身、竖构图。
  - 平涂/赛璐璐上色，干净线稿，不要多余描边、不要像素马赛克。
  - 建议高 ≥ 800px（程序按比例自适应缩放，越清晰越好）。
  - 六张尺寸/构图（人物大小、位置）尽量对齐，避免切换时跳动。

【推荐工具】
  - 动漫大模型：NovelAI / Stable Diffusion(AnimagineXL 等) / 即梦 / Liblib / 吐司。
  - 立绘差分流水线：ComfyUI_VNCCS、TavernSprite（一张母版批量出多表情）。
  - 抠透明底：remove.bg / Photoroom / PS 魔棒。

【合规】用「气质参考 + 原创化」提示词，避免 1:1 还原既有 IP；
        角色名集中在 core/persona.py::PERSONA_NAME，正式版改一处即可。
