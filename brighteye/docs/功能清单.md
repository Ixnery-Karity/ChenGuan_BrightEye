## 一、基础监测功能

### 1.1 视觉感知（`vision/`）
| 功能 | 说明 | 实现 | 版本 |
| --- | --- | --- | --- |
| 眨眼频率监测 (EAR) | FaceMesh 6 点眼纵横比，连续低阈值帧判定一次眨眼；<10 次/分提示干眼风险 | `vision/detectors.py` + `core/blink_counter.py` | v1.0，**v1.16 修正快速连眨读数虚高**（60s 窗+外推下限） |
| 用眼距离估算 | `distance = focal × 平均瞳距 / 像素瞳距`，45cm 防线告警 | `vision/detectors.py` | v1.0 |
| 坐姿体态监测 (CVA) | Pose 关键点**同侧拓扑**算法（左耳配左肩/右耳配右肩、可见度加权），判定低头/前倾/高低肩 | `vision/detectors.py` | v1.0，v1.9 换同侧拓扑 |
| 表情情绪识别 | 52 维 blendshapes → FACS 动作单元 → Ekman 情绪原型映射 + EMA 平滑 + 迟滞去抖，输出 积极/平静/疲惫/压力/低落；零新增模型、可解释无需训练 | `core/emotion.py` | v1.8 |
| 多进程视觉推理 | 采集+推理搬独立子进程（spawn 安全），Queue 回传轻量样本，规避 GIL 掉帧；崩溃自动回退 | `vision/worker.py`，`--mp-vision` | v1.9 |
| 无摄像头模拟器 | 合成数据后端（含渐趋疲惫表情系数），任何机器可完整演示全链路 | `core/simulator.py` | v1.0 |
| 视觉后端后台加载 | mediapipe 导入+摄像头+双模型加载移入后台线程，UI 秒开热切换 | `core/monitor.py` | **v1.11** |

## 附：交叉支撑能力

- **统一 LLM 客户端**（`core/llm_client.py`，v1.8）：仅标准库 urllib，自动探测 OpenAI 兼容 API → 本地 Ollama → 离线兜底；三大方向的 AI 能力共用此入口。
- **电竞导播风仪表盘 + 粒子背景自适应降载**（`ui/app.py` + `ui/particles.py`）。
- **一键安装包**（`tools/build_exe.py`，v1.9，**v1.15 全自动**）：PyInstaller onedir
  打完自动定位 ISCC 编译 Inno Setup，直接产出中文安装向导 Setup.exe；
  免管理员安装，安装版数据落 `%LOCALAPPDATA%\ChenguanBrightEye`；
  **v1.16 补双击卸载入口**（开始菜单卸载项 + 卸载时询问是否删用户数据，默认保留）。
- **环境一键安装**（`安装运行环境.bat` + 启动器探测，**v1.16**，**v1.17 防版本冲突**）：
  双击启动器自动检测 Python/依赖缺失并弹窗一键安装（winget/官网下载装 Python 3.12，
  pip 清华镜像兜底），全新机器源码也能双击即用；v1.17 只接受 Python 3.10~3.14、
  选定解释器绝对路径固定入 `python_path.txt`，多版本共存不串、新旧版本零冲突。
- **安装包大模型集成**（`tools/build_exe.py` + `llm_models/install_models_setup.bat`，**v1.17**）：
  安装向导可勾选随装 Ollama + 聊天/分析双模型；任一环节失败自动回退删除已拉模型并
  弹窗提醒（app 本体不受影响，自带 Python 运行时）；卸载删模型与用户数据，
  保留 Python 环境与 Ollama 程序（共享组件不误伤）。
- **GitHub Release 制**（**v1.17 起**）：每版 git tag + Release 附源码 zip/安装包，
  历史版本可补发 Release，仓库不做整体替代。
- **加载页介绍轮播**（`ui/splash.py`，**v1.16**）：等待期间轮播 7 条项目卖点
  与弥悠性格台词（4s/条），首启等待不再无聊，兼作项目「思想钢印」宣传位。
- **版本规范与路线图**（`docs/版本规范与路线图.md`，**v1.16**）：语义化版本
  规范 + 历史脉络梳理 + 1.x/2.x/3.x 未来规划；发版四件套追加 git tag。
- **系统托盘图标**（`ui/tray.py`，**v1.15**）：右下角通知区域常驻——左键开仪表盘、
  右键菜单（四模式打勾切换/聊天/退出）；纯 ctypes Shell_NotifyIcon 零新依赖，
  explorer 重启自动补挂，非 Windows 静默降级。
- **启动秒开/退出秒退**（`ui/pet.py` + `ui/app.py`，**v1.15**）：立绘后台异步
  「缩略图→抠图→热切换」（旧同步 floodfill 需 30~90s），加载页点开即见；
  退出收尾（报告保存/摄像头释放）后台化，窗口秒消失。
- **git 版本控制 + GitHub 开源**（v1.10）：`Ixnery-Karity/ChenGuan_BrightEye`。
