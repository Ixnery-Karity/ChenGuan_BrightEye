# 打包发布指南 · 一键安装包（建议2）

> 目标：让评委/用户**免装 Python 环境**双击即用。
> 工具链：PyInstaller（onedir）→ Inno Setup（安装向导 setup.exe）。

## 一、快速开始

```bash
# 1. 安装打包器（一次性）
pip install pyinstaller

# 2. 在 challenge 目录执行一键打包
python -m brighteye.tools.build_exe

# 产物
dist/宸观BrightEye/宸观BrightEye.exe      # 免环境直接运行（整个文件夹分发）
build_installer.iss                       # Inno Setup 脚本（下一步）
```

## 二、生成安装向导 setup.exe（可选）

1. 安装 [Inno Setup 6](https://jrsoftware.org/isinfo.php)（免费）；
2. 双击项目根目录的 `build_installer.iss` → 菜单 Build → Compile；
3. 产出 `Output/宸观BrightEye_Setup_v1.9.0-demo.exe`——带简体中文安装向导、
   开始菜单/桌面快捷方式、卸载入口。

## 三、关键打包参数说明（build_exe.py 已内置）

| 参数 | 作用 |
|---|---|
| `--windowed` | 无控制台黑窗（GUI 应用） |
| `--add-data brighteye/assets` | 模型 `.task` + 文乃立绘随包 |
| `--collect-all mediapipe` / `cv2` | 收齐二进制依赖与运行时数据文件 |
| `--hidden-import brighteye.vision.worker` | 多进程(spawn)子进程入口必须显式打入 |
| onedir（默认，非 onefile） | mediapipe 动态库多，onedir 启动快且稳定 |

## 四、瘦身与替代方案

- **体积**：全量约 400~600MB（mediapipe+opencv 占大头）。可换
  `opencv-python-headless` 省 ~80MB（本项目不用 cv2 GUI）。
- **Nuitka**（进阶）：`python -m nuitka --standalone brighteye/main.py`，
  编译为原生代码，启动更快、更难反编译，但编译耗时长（>30min），
  适合正式发布而非迭代期。
- **仅演示不打包**：评委机器有 Python 时，
  `pip install -r requirements.txt && python -m brighteye.main --simulate`
  依旧零门槛（离线铁律路径）。
