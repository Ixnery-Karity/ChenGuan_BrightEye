# 打包发布指南 · 一键安装包（建议2，v1.15.0 全自动）

> 目标：让评委/用户**免装 Python 环境**双击即用。
> 工具链：PyInstaller（onedir）→ Inno Setup（安装向导 setup.exe）。
> v1.15.0 起两步合一：build_exe.py 打完 exe 自动定位 ISCC 编译出安装包。

## 一、快速开始

```bash
# 1. 安装工具链（一次性）
pip install pyinstaller
winget install -e --id JRSoftware.InnoSetup    # 或到官网下载安装

# 2. 在 challenge 目录执行一键打包（约 5~15 分钟）
python -m brighteye.tools.build_exe

# 产物
dist/宸观BrightEye/宸观BrightEye.exe                    # 免环境直接运行（绿色文件夹）
build_installer.iss                                     # Inno Setup 脚本（自动生成）
dist_installer/宸观BrightEye_Setup_v1.15.0-demo.exe     # ★ 中文安装向导（最终成品）
```

## 二、安装包行为（build_installer.iss 已内置）

- **免管理员**：`PrivilegesRequired=lowest`，默认装到
  `%LOCALAPPDATA%\Programs\ChenguanBrightEye`；想装 Program Files 时
  向导会自行请求提权。
- 自动创建**开始菜单 + 桌面快捷方式**（带 Q 版弥悠图标）与**卸载入口**；
  安装完成页可勾选「立即启动」。
- **用户数据落位**：安装版（frozen）运行时 `data/`、`reports/` 自动改锚
  `%LOCALAPPDATA%\ChenguanBrightEye\`（安装目录通常不可写，config.py 已适配），
  卸载重装不丢历史档案。
- 未装 Inno Setup 时 build_exe.py 只生成 .iss 并提示（不算失败）：
  装好后双击 `build_installer.iss` → Build → Compile 即可手动编译。

## 三、关键打包参数说明（build_exe.py 已内置）

| 参数 | 作用 |
|---|---|
| `--windowed` | 无控制台黑窗（GUI 应用） |
| `--add-data brighteye/assets` | 模型 `.task` + 文乃立绘随包 |
| `--collect-all mediapipe` / `cv2` | 收齐二进制依赖与运行时数据文件 |
| `--hidden-import brighteye.vision.worker` | 多进程(spawn)子进程入口必须显式打入 |
| onedir（默认，非 onefile） | mediapipe 动态库多，onedir 启动快且稳定 |

## 四、瘦身与替代方案

- **体积**：全量约 400~600MB（mediapipe+opencv 占大头）。可换
  `opencv-python-headless` 省 ~80MB（本项目不用 cv2 GUI）。
- **Nuitka**（进阶）：`python -m nuitka --standalone brighteye/main.py`，
  编译为原生代码，启动更快、更难反编译，但编译耗时长（>30min），
  适合正式发布而非迭代期。
- **仅演示不打包**：评委机器有 Python 时，
  `pip install -r requirements.txt && python -m brighteye.main --simulate`
  依旧零门槛（离线铁律路径）。
